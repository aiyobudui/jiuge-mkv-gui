# 九歌 MKV 混流工具

一个基于 Python + PySide6 开发的 MKV 视频混流工具，支持批量添加字幕、音轨、封面附件，并设置默认轨道。

## 功能特点

- **批量视频处理** - 支持同时处理多个视频文件
- **字幕管理** - 添加外部字幕文件，自动按序号匹配视频
- **音轨管理** - 添加外部音轨文件，自动按序号匹配视频
- **附件管理** - 添加封面图片，自动嵌入到视频文件
- **默认轨道设置** - 可设置默认音轨和默认字幕
- **轨道选择** - 可选择保留或删除视频中的原有字幕/音轨
- **输出格式选择** - 支持 MKV 和 MP4 两种输出格式
- **CRC 校验** - 可选添加 CRC 校验和
- **多线程处理** - 支持多线程并行混流，提高处理效率

## 界面预览

### 视频选项卡
- 添加视频文件到处理队列
- 显示视频文件大小、字幕轨数量、音轨数量
- 支持拖拽添加文件
- 多线程读取轨道信息，快速加载大量文件

### 字幕选项卡
- 选择字幕源文件夹
- 自动按序号匹配视频（第1个字幕→第1个视频）
- 显示字幕文件列表和匹配状态

### 音轨选项卡
- 选择音轨源文件夹
- 自动按序号匹配视频（第1个音轨→第1个视频）
- 显示音轨文件列表和匹配状态

### 附件选项卡
- 选择封面图片文件夹
- 自动将封面嵌入到所有视频文件
- 支持 JPG、PNG、GIF、WebP 等图片格式

### 开始混流选项卡
- 设置输出目录
- 选择输出格式（MP4/MKV）
- 配置混流选项（保留字幕/音轨、默认轨道等）
- 任务队列管理
- 进度显示

## 系统要求

- Windows 10/11
- [MKVToolNix](https://mkvtoolnix.download/) (需要安装并设置 mkvmerge.exe 路径)

## 安装使用

### 方式一：从源码运行

```bash
# 克隆仓库
git clone https://github.com/aiyobudui/jiuge-mkv-gui.git
cd jiuge-mkv-gui

# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

### 方式二：从源码编译

```bash
# 克隆仓库
git clone https://github.com/aiyobudui/jiuge-mkv-gui.git
cd jiuge-mkv-gui

# 一键编译（推荐）
build.bat

# 或手动编译
pip install -r requirements.txt
python -m PyInstaller --windowed --name "jiuge-mkv-gui" main.py
```

编译完成后，可执行文件位于 `dist/jiuge-mkv-gui/` 目录。

## 使用说明

### 1. 添加视频文件
在"视频"选项卡中，点击"浏览"选择视频文件夹，或直接拖拽文件到列表中。

### 2. 添加字幕文件
在"字幕"选项卡中，点击"浏览"选择字幕文件夹。字幕文件会自动按序号匹配视频：
- 字幕列表第1个文件 → 视频列表第1个视频
- 字幕列表第2个文件 → 视频列表第2个视频
- 以此类推...

### 3. 添加音轨文件
在"音轨"选项卡中，操作方式与字幕相同。

### 4. 添加封面附件
在"附件"选项卡中，点击"浏览"选择封面图片文件夹。封面会自动添加到所有视频文件中。

### 5. 设置混流选项
在"开始混流"选项卡中：
- **输出目录**：选择混流后文件的保存位置
- **输出格式**：选择 MP4 或 MKV 格式
- **轨道选择**：点击"轨道选择"按钮可选择保留哪些字幕/音轨，设置默认轨道
- **外部字幕设为默认**：一键将所有外部字幕设为默认
- **CRC校验**：在文件名中添加CRC校验和
- **移除旧CRC**：移除原文件名中的CRC校验和
- **保留日志**：保留混流日志文件
- **出错中止**：遇到错误时停止处理

### 6. 开始混流
点击"添加到队列"将任务添加到队列，然后点击"开始混流"开始处理。
